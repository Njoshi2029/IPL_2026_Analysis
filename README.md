
# 🏏 IPL 2026 Cricket Analytics

## 📌 Project Overview

This project analyzes **IPL 2026 ball-by-ball data** using Python to gain insights into team and player performance. It includes data cleaning, exploratory data analysis (EDA), feature engineering, and visualizations to understand scoring patterns, batting, bowling, and match phases.

## 🎯 Objectives

* Analyze IPL 2026 match data
* Perform data preprocessing and cleaning
* Explore batting and bowling performance
* Compare team statistics
* Visualize key cricket insights

## 🛠 Technologies Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib

## 📂 Dataset

* IPL 2026 Ball-by-Ball Dataset

## 📊 Key Analysis

* Team-wise total runs
* Top run scorers
* Top wicket takers
* Boundary analysis (4s & 6s)
* Powerplay, Middle Overs & Death Overs analysis
* Over-wise scoring trends
* Economy rate and strike rate analysis

## 📈 Features

* Data Cleaning & Preprocessing
* Feature Engineering
* Exploratory Data Analysis (EDA)
* Cricket Performance Analysis
* Data Visualization

## 🚀 How to Run

1. Clone the repository.
2. Install the required libraries.
3. Open `IPL_2026_analysis.ipynb` in Jupyter Notebook.
4. Run all cells.
